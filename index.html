Introduction

Kubuntu is the official flavor of Ubuntu with the KDE 5 Plasma desktop environment. KDE is one of the modern desktop environment on Linux. It is really fast and clean. The KDE 5 Plasma desktop environment runs on low memory and works pretty great.

Why try Kubuntu

There are many good reasons to install Kubuntu, such as:

  • Kubuntu makes your PC easier to use.
  • Kubuntu is an operating system built by a friendly worldwide team of expert developers. It contains all the applications you need, including a web browser, office suite, media apps, instant messaging, in addition to many others.
  • Kubuntu is a free, open-source alternative to Windows and Mac OS X.
  • It can be easily installed alongside, or instead of, Windows
  • Extensive community support.

Downloading Kubuntu

An iso image of Kubuntu can be downloaded from official website

Verify the iso image

Check the MD5SUM to ensure that your download is complete and that it has given you a fully working ISO to boot, which can be later installed from.

From Windows XP -> 7 -> 8 -> 10

This utility provided by microsoft can be used to verify checksum

From Mac OS X

Open the Terminal app, then navigate to the directory where the ISO is saved. Run:

md5 -r name-of-iso.iso

And compare the output to the hashes page for the ISO file.

From Ubuntu Linux

The first thing to do is to make sure you're in the directory with the .iso file (It will most likely be found in the Downloads directory). Then run the command:

md5sum kubuntu-iso-name.iso

Making a Bootable USB Stick

On Windows, Rufus can be used to make bootable Kubuntu 18.04 LTS USB thumb drive.

On linux go to the terminal and run the follwing command:

$ sudo dd if=/path/to/kubuntu.iso of=usb-name bs=1M
On next step we'll focus on installing kubuntu

Installing Kubuntu

Most computers will boot from USB automatically. Simply insert the USB flash drive and either power on your computer or restart it. You should see the welcome window prompting you to choose your language and either install or try the Ubuntu desktop.

If your computer doesn't automatically boot from USB, try holding F12 when your computer first starts. With most machines, this will allow you to select the USB device from a system-specific boot menu.

  1. The Kubuntu installer should start. Now select the language and click on Continue.
  2. Now select keyboard layout and click on Continue.
  3. Now you can either select Minimal installation to install minimal version of Kubuntu or Normal installation to install the full version of Kubuntu. You can also check Install third-party software for graphics and Wi-Fi hardware and additional media formats to install third part drivers and media codecs. It requires internet connectivity.   4. Now you can select partition method. For new users, or those who don't want to dual boot, you can check guided-use entire disk.
  5. Now select timezone and click on Continue.
  6. Now type in your user information and click on Continue. The installation should start.

Conclusion

Once the installation is complete click on restart now. You PC should reboot.

After a new installation go to terminal and type sudo apt update && sudo apt upgrade --y to update all softwares. You can also install any software you want via terminal. For example: sudo apt install firefox. To remove an app type sudo apt remove firefox.
